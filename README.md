# TruckInfo

Aplicación móvil para gestión de transporte y seguimiento de camiones.

## Características

- Login con autenticación
- Panel de información de camiones
- Gestión de envíos y viajes
- Configuración de usuario
- Interfaz moderna con React Native y Expo

## Instalación

1. Clonar el repositorio
2. Instalar dependencias:
   ```bash
   npm install
   ```

3. Iniciar la aplicación:
   ```bash
   npx expo start
   ```

## Estructura del Proyecto

- `app/` - Componentes principales y navegación
- `services/` - Servicios de API y datos
- `components/` - Componentes reutilizables

## Tecnologías

- React Native con Expo
- TypeScript
- Expo Router para navegación
